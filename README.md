# GifApp 

Repositorio del proyecto de GifApp en react con yarn :D

## Comandos de Yarn

Para instalar las dependencias del proyecto:
```bash
yarn install
```

Para iniciar el servidor de desarrollo:
```bash
yarn start
```

Para crear una versión de producción:
```bash
yarn build
```

Para ejecutar las pruebas:
```bash
yarn test
```
